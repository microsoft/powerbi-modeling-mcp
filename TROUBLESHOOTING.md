# Troubleshooting

This guide helps you diagnose and resolve common issues with the Power BI Modeling MCP Server. 

## Check the output window for more details.

Whenever there is an issue with MCP server in **Visual Studio Code** it will be logged in the [**Output**](https://code.visualstudio.com/docs/copilot/customization/mcp-servers#_mcp-output-log) window. You can click **Show Output** to open the MCP output window.
